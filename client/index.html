Iroh Tunnel

Request Sandstorm capabilities, save them into the grain, and verify that they can be restored on demand. This is the raw UiView baseline for the tunnel app.

RawUdp Path

This view is optimized for one task: bind Sandstorm RawUdp, exchange tickets, and probe the iroh connection over the custom transport.

  1. Click Request IP Interface and save the capability.
  2. In Saved Capabilities, click Use for RawUdp.
  3. Wait until the ticket below contains a custom: line.
  4. Copy each grain’s full local ticket into the other grain and click Probe Connection.
